# Informática para la investigación: Simuladores interactivos

Simuladores HTML autónomos publicados en GitHub para el curso de Informática para la investigación (UNAV/Tecnun, junio 2026).

| Carpeta          | Archivo Principal  | Descripción                                                                                                                               |
| ---------------- | ------------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
| `simuladores_1/` | `delta_radiomics.html` | Analizador interactivo de delta-radiomics: carga datos de MR-Linac, calcula Δ% por feature y clasifica la respuesta tumoral al tratamiento |
| `simuladores_2/` | `roc_curve.html` | Curva ROC interactiva: ajusta el umbral de decisión con un slider y observa cómo cambian sensibilidad, especificidad y la matriz de confusión en tiempo real. |
| `simuladores_3/` | `kfold_validation.html` | Validación K-fold: visualiza cómo se dividen los datos en K subconjuntos y cómo rota el fold de validación en cada iteración. |
| `simuladores_4/` | `rt_fractionation.html` | Fraccionamiento RT: calcula BED y EQD2 con el modelo lineal-cuadrático variando número de fracciones y dosis, comparando con RT convencional. |
| `simuladores_5/` | `radiomic_features.html` | Extracción de features radiómicas: edita una imagen 10×10 píxeles y observa el cálculo en tiempo real de Mean Intensity, Entropía, Contraste y otras features. |
